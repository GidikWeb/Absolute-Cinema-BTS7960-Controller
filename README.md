# Absolute-Cinema-BTS7960-Controller

## 📌 О проекте

**D-LEEK** — это открытая, недорогая и функциональная альтернатива дорогостоящим реабилитационным аппаратам (таким как R-Sleek или LPG). Устройство предназначено для медицинской реабилитации после травм, неврологических заболеваний, переломов, а также для восстановительной терапии в домашних условиях.

### 🎯 Ключевые особенности

- **Низкая стоимость** — менее 15 000 рублей (в 100+ раз дешевле аналогов)
- **Профессиональный функционал**:
  - Плавный разгон и торможение (Ramping)
  - Защита по току (Overcurrent protection)
  - Таймер безопасности непрерывной работы
  - 9 предустановленных программ массажа
  - Редактор пользовательских программ (до 5 программ, 10 шагов каждая)
- **Интуитивный интерфейс** — LCD 16×2 с кастомными иконками
- **Управление** — 3 кнопки (Up/OK/Down) + Serial команды
- **Автономность** — все настройки сохраняются в EEPROM

## ⚙️ Аппаратная платформа

| Компонент               | Модель / Спецификация                          |
|------------------------|------------------------------------------------|
| Микроконтроллер        | Arduino Uno (или совместимый)                  |
| Драйвер двигателя      | BTS7960 (43A пик, 27V max)                     |

## 🎮 Управление

### Через кнопки
- **UP** — увеличение значения / навигация вверх
- **DOWN** — уменьшение значения / навигация вниз / остановка
- **OK** — подтверждение / вход в меню / пауза/возобновление

### Через Serial (115200 бод)

| Команда                         | Описание                                      |
|--------------------------------|-----------------------------------------------|
| `ON` / `OFF`                   | Включить / выключить мотор                    |
| `SPEED 150`                    | Установить скорость (0-255)                   |
| `DIR F` / `DIR R`              | Направление: Forward / Reverse                |
| `TIMER 10`                     | Установить таймер (минут)                     |
| `START_TIMER` / `STOP_TIMER`   | Запуск / остановка таймера                    |
| `SAFETY 30`                    | Тайм-аут безопасности (мин)                   |
| `LIST`                         | Список всех программ                          |
| `RUN 2`                        | Запустить программу по индексу                |
| `STOP` / `PAUSE` / `RESUME`    | Управление программой                         |
| `PROG_INFO 0`                  | Показать шаги программы                       |
| `EDITOR_NEW`                   | Создать новую программу                       |
| `EDITOR_ADD_STEP 150 1 30`     | Добавить шаг (скорость, направление, секунды) |
| `EDITOR_SET_NAME MyProg`       | Задать имя программы                          |
| `EDITOR_SAVE`                  | Сохранить программу в EEPROM                  |
| `AUTO_TEST`                    | Запустить автоматическое тестирование         |
| `HELP`                         | Показать все команды                          |

## 📦 Предустановленные программы массажа

| №  | Название       | Особенность                                  |
|----|----------------|----------------------------------------------|
| 0  | Hard           | Интенсивный, длительные шаги                 |
| 1  | Relax          | Мягкий, расслабляющий                        |
| 2  | Body           | Общеукрепляющий                              |
| 3  | Arms           | Для рук и плечевого пояса                    |
| 4  | Legs           | Для ног                                      |
| 5  | Kneading       | Разминание с переменной скоростью            |
| 6  | Vibration      | Быстрое чередование направлений              |
| 7  | Tapping        | Имитация постукивания (вкл/выкл)             |
| 8  | Lymphatic      | Длительный лимфодренаж (60 мин)              |

> Пользовательские программы сохраняются в EEPROM (до 5 штук, до 10 шагов).
